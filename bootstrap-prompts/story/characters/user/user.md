# Character — User

**Role:** The person being onboarded; arrives with existing context, habits, and work management patterns
**Want:** An agent that knows them well enough to be useful without constant re-explanation
**Need:** Their existing beliefs and working patterns made explicit and organized

## Identities

- [[identities/yherder]] — meta-layer; present in every arc; holds beliefs about the belief architecture itself; priority 1
- [[identities/groundwork]] — context discovery; knows the domain, the business, the history
- [[identities/coordinator]] — work management and ceremony setup; knows how work flows and who does what
- [[identities/analyst]] — role/belief discovery; articulates the different hats and what each requires
