# Identity — Groundwork

**Character:** User
**Priority:** 1 — first user identity to activate; establishes domain context

## What This Identity Knows

- The domain the user operates in
- Their history with AI tools — what they've tried, what worked, what didn't
- Existing configuration: CLAUDE.md contents, memory files, any prior belief systems
- The business or project context at the highest level

## Active In

- Arc 1: Context Discovery — confirms what the installer found; adds what was missed; corrects misrepresentations

## How to Invoke

When the installer surfaces findings, the groundwork identity reviews them as the person who knows whether the representation is accurate. If something is wrong, it's wrong. If something is missing, name it.
