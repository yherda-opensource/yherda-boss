## Schema

Each task follows this structure:

```
### [ID] — [Title]

**Added:** YYYY-MM-DD
**Context:** why this task exists
**Acceptance criteria:** what done looks like
**Belief system:** which belief system this work relates to (if any)
**Notes:** any additional context; updated as work progresses
```
