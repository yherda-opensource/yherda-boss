# Retro Activity Log

Append-only audit trail of every belief transition produced through retro. Do not edit existing entries — add new ones at the bottom.

---

## Schema

Each entry follows this structure:

```
### YYYY-MM-DD — {Belief statement}

**Belief system:** which belief system this was integrated into
**Previous status:** none | emerging | active | strained | former
**New status:** emerging | active | strained | former
**Retro session:** which retro surfaced this
**Reason:** why this transition was made
**Evidence:** what supported the decision
**SOS nominated:** yes | no
```
